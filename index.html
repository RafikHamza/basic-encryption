Encryption Basics

Caesar Cipher

A substitution cipher where each letter is shifted a fixed number of positions in the alphabet.

Example:

Plaintext: HELLO, Key: 3

Ciphertext: KHOOR

A B C D E F ...
A B C ...

XOR Cipher

A cipher that applies the XOR logical operation between plaintext bits and key bits.

Example:

Plaintext: ABC, Key: KEY

Binary XOR operation shown below:

A (65): 01000001
K (75): 01001011
XOR:    00001010 → 
 (control char)

Vigenère Cipher

A method of encrypting text using a series of Caesar ciphers with different shift values based on a keyword.

Example:

Plaintext: HELLO, Key: KEY

K shifts by 10, E shifts by 4, Y shifts by 24

Result: RIJVS
